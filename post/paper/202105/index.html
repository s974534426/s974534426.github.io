1. Discrete Adversarial Attacks and Submodular Optimization with Applications to Text Classification

文本输入的对抗样本任务的难点是无法使用像图像攻击那样的通过修改某几个像素计算相关像素的梯度的方法；在替换单词后如何保证语义、语法的正确性，并且不会替换太多的单词(改动尽量微小)；搜索空间大。

本文是关于NLP中的对抗样本，将该问题形式化为子模优化，并使用梯度+贪心的方法求解，是白盒攻击，测试任务是对文档的分类，在句子和单词级别同时进行攻击，需要关注的问题是：如何形式化子模优化问题，如何选择搜索空间，如何涉及高效算法。

  1. 问题形式化：只允许最多修改$m$个特征(使用l0范数限制$||l||_0 \leq m$)，$l^* = argmax _{||l||_0 \leq m}C_y(V(T_l(x)))$，$T_l(x)$表示将输入x替换为对抗样本的一种替换方式，$V$将离散样本转化为embedding，$C_y$表示y的得分，所以这个式子的目标是找到一个替换$l^*$使得y的得分最高。定义集合函数$f(S)=\max_{supp(l)\subset S}C_y(V(T_l(x)))$，则优化问题为$S^* = argmax_{|S|\leq m}f(S)$，$S$的每个元素是原句子的一种可行的替换方式(即一个对抗样本)。

    一般的研究中都会对函数$f$做出一些假设，比如光滑或子模。而该问题中子模函数$f(S)$是非递减的(文中没有严格证明)，然后证明了简化版的WCNN和RNN是子模函数(要求过多)，

  2. 搜索空间，搜索空间是由释义(paraphrasing)确定每个词和句子的候选替换(应该就是同义词和同义句的替换吧)。利用Word Mover Distance判断语义的相似性，用语言模型判断语法的正确性，筛选出符合条件的对抗样本。

  3. 优化算法：使用梯度引导的贪心算法。首先以句子为单位划分文档，然后以句子为单位逐句子贪心替换；然后以词为单位划分文档，通过embedding将句子转化为向量，然后计算在每个单词位置的梯度，将梯度最大的N个位置的单词依次替换为搜索空间中所有的候选单词，这部分是结合梯度和贪心的搜索。

实验上与另外的一篇使用相同网络的文章进行了比较。

不足：1是所有的实验都是在两个简单的网络上进行，并且只和一篇论文进行了比较，并不够充分。但这也是一个共性的问题：在复杂模型上，无法证明网络函数是子模的，只能通过实验得出使用子模优化算法和比较好效果的baseline差的不多，在简单模型上，实验又不够充分。2是对抗训练的评估应该怎么评估，使用方法A进行对抗训练，又使用A再次攻击，发现攻击成功率变低，能否说明对抗训练是有效果的呢。

2. Parsimonious Black-Box Adversarial Attacks via Efficient Combinatorial Optimization

对图片的黑盒攻击，使用子模优化的方式求解。

不足：无法证明神经网络的近似子模性，只通过实验说明使用贪心求解算法或Local Search和PGD效果差不多这一实验得到神经网络函数满足近似子模性是不够的。

  1. 问题形式化：将图像添加随机扰动的问题，形式化为像素的选择问题。

3. Adversarial Examples For Natural Language Classification Problems

这篇文章是1的基础，应该是NLP对抗样本中一篇比较早的工作，1的工作主要是对该文章内容的改进。大部分内容在1中都有提及。

  1. 问题形式化：$\max_{x'} J(x'), s.t. ||x-x'|| \leq \epsilon$。对于约束条件中的限制具体到该问题是语义和语法的差异性不能过大，本文对语义和语法的相似性度量提出了两种方法，对语义，使用thought vector，对语法使用language model。

  2. 算法：贪心替换。只在word level进行了替换，具体方法是，对于句子中每一个词贪心的尝试使用可行的同义词进行替换，然后在替换后的所有句子中，找到一个使得优化目标最大的，作为下一轮的输入。
