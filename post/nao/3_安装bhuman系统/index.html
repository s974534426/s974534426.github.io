3. 安装BHuman系统

2020-10-10
NAO

文章目录

    • 1. 安装软件
    • 2. 添加机器人

1. 安装软件

首先安装操作系统，建议使用Ubuntu14.04，与文档保持一致。因为安装更高版本的系统的话，使用apt下载的包版本与原来使用的版本不一致，代码会无法编译，比如Qt的高版本删除了很多低版本的函数，导致不能编译，这个如果配置的话也会比较复杂，所以建议使用虚拟机安装Ubuntu14.04。

下载BHuman相关代码，安装依赖，执行文档中的sudo apt-get install ...命令。

然后再手动安装alcommon，需要先在官网下载naoqi-sdk-2.1.4.13-linux32.tar.gz，然后执行Install/installAlcommon脚本。

之后进入Make/Linux下执行make命令，进行编译。

2. 添加机器人

执行./createRobot -t 1 -r 5 Penny，创建一个队伍号为1，队员编号为5的名叫Penny的机器人，队伍号是之后的IP地址的第三位，队员号是之后IP地址的第四位，比如上面的机器人之后的有线IP地址会是192.168.1.5，之后通过这个IP访问。注意队伍号建议设置为1，队员号设置为一个比较大的数，队伍号不是1时，似乎会出现一些无法连接的问题，队员号如果比较小，可能会发生冲突。

之后在开机的情况下执行./addRobotIds -ip 192.168.1.101 Penny，这里的IP是机器人当前的IP，不是上面的192.168.1.5。

之后再打开Build目录中的bush，就可以看到新建的机器人，并且如果可以看到电量和延迟的话，说明已经连接上了机器人。或者使用ssh，ping等命令，检查IP是否有效，如果可以登录或ping有反应，说明已经添加成功。

注意上面每一步执行后，都会有提示，如果没有提示，可能是某个配置文件出了问题，比如之前手动删除了某个robot，导致文件内有语法错误(这些bug不会显示出来，但如果没有执行成功，就说明有bug)。
